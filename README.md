# ShortestPathFinder

Этот проект реализует алгоритм поиска кратчайших путей в графе с использованием алгоритма поиска в ширину (BFS). Граф считывается из файла, и для каждой вершины выводятся её расстояния от заданной начальной вершины.

## Как использовать

1. Клонируйте репозиторий:

    ```bash
    git clone https://github.com/Lefthander07/ShortestPathFinder.git
    ```

2. Перейдите в директорию проекта:

    ```bash
    cd ShortestPathFinder
    ```

3. Создайте и перейдите в директорию для сборки:

    ```bash
    mkdir build
    cd build
    ```

4. Выполните команду CMake для генерации файлов сборки:

    ```bash
    cmake ..
    ```

5. Соберите проект с помощью `make`:

    ```bash
    make
    ```

6. Запустите программу:

    После сборки, вы можете запустить программу, используя скомпилированный исполнимый файл в директории `build`.

    ```bash
    ./ShortestPathFinder
    ```

## Описание

### Оператор перемещения

Оператор перемещения и конструктор перемещения были реализованы для улучшения производительности при работе с большими графами. В случае, если граф содержит большое количество вершин и рёбер, копирование данных может занять значительное время и ресурсы. Перемещение позволяет эффективно передавать владение данными между объектами без необходимости их копирования, что значительно ускоряет работу программы и уменьшает использование памяти.

### Алгоритм BFS

Алгоритм **поиска в ширину** (BFS) используется для поиска кратчайших путей от начальной вершины ко всем остальным вершинам в графе. Он работает за время O(V + E), где V — количество вершин, а E — количество рёбер.

## Структура проекта

- **Graph.h** — заголовочный файл с объявлением класса `Graph` и его методов.
- **Graph.cpp** - файл программы с определением методов класса `Graph`
- **main.cpp** — главный файл программы, который выполняет чтение графа из файла, запуск алгоритма BFS и вывод расстояний.
- **CMakeLists.txt** — файл конфигурации для сборки проекта с использованием CMake.
